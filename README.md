# 💊 Medication Reminder Bot

Telegram bot for medication reminders with confirmation system.

## Features
- ✅ Medication scheduling
- 🔔 Automatic reminders  
- 👍 Confirmation with praise
- 💾 SQLite database
- 🐳 Docker support

## Deployment
Deployed on [Railway](https://railway.app)

## Tech Stack
- Python 3.11
- python-telegram-bot
- APScheduler
- SQLite
